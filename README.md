# GifSprite

A fun animated sprite for your JetBrains IDE! It moves when you type!

## Features

- **Drag & Drop** - Move the sprite anywhere you want
- **Resize** - Make it bigger or smaller (0-512)
- **Two Modes** - "Type Mode" (moves when you type) or "Auto Mode" (plays all the time)
- **Speed Control** - Make it faster or slower
- **Custom GIFs** - Use your own GIF files
- **Any Language** - Name your GIFs in Chinese, Japanese, etc.

## How to Install

Install from JetBrains Marketplace (coming soon)

## How to Use

1. After install, you'll see the sprite in the bottom-right corner
2. **Drag** it to move it around
3. **Settings** → Appearance → GifSprite to change options:
   - Size: Make it bigger or smaller
   - Mode: Auto Play or Type Triggered
   - Speed: Faster or slower
   - Import GIF: Add your own GIF
4. **View** → Toggle GifSprite to show/hide

## Settings

| Option | What it does |
|--------|-------------|
| Enable | Show or hide the sprite |
| Size | How big (0-512) |
| Mode | Auto Play = always moving, Type Triggered = moves when you type |
| Speed | Lower number = faster (10-500) |
| Sprite Pack | Pick which GIF to use |
| Import GIF | Add your own GIF file |
| Reset | Put sprite back to start position |
