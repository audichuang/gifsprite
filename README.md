# ZipSprite

A JetBrains IDE plugin that displays a cute animated sprite that moves as you type!

## Features

- Draggable sprite overlay on your IDE
- Customizable size (0-512 DIP)
- Smooth 24-frame animation triggered by typing
- Toggle visibility from View menu
- Settings in Appearance preferences

## Installation

Install from the JetBrains Marketplace (coming soon)

## Usage

1. After installation, the sprite will appear in the bottom-right corner of your IDE
2. Drag it anywhere you like
3. Adjust size in Settings > Appearance > ZipSprite
4. Toggle visibility via View menu > Toggle ZipSprite
